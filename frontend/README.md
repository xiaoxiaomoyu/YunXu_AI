# YunXu AI Frontend

Vue 3 + Vite frontend for YunXu AI.

## Scripts

```bash
npm install
npm run dev
npm run build
```

The API base URL is currently configured in `src/services/api.js` as `http://localhost:8080`.

## Main Views

- `/`：应用中心
- `/ai-chat`：通用 AI 对话
- `/game`：哄哄模拟器
- `/customer-service`：智能客服
